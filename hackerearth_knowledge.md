## Product Suite (The Complete Hiring & Engagement Suite)

### SOURCE
1. **Job Posting** — AI-enhanced job posting engine with semantic matching, distributed across HackerEarth ecosystem
2. **Hiring Challenges** — Managed hackathon-style hiring events, CS team + marketing support, AI evaluation

### SCREEN
3. **AI Screener** — Autonomous AI agent that replaces manual resume review and early phone screens; structured, bias-resistant insights
4. **Technical Assessment** — 40,000+ problems, 1,000+ skills, 40+ languages, real-world project simulations, AI evaluates code quality/logic/efficiency
5. **Soft Skill Assessment** — Behavioral and communication evaluation

### INTERVIEW
6. **AI Interviewer (OnScreen)** — End-to-end AI-conducted technical + behavioral interviews, 24/7, lifelike AI avatars, identity verification, 25,000+ deep technical questions
7. **FaceCode** — Live coding interview platform: integrated IDE, pair-programming, AI-assisted insights, advanced proctoring

### ENGAGE
8. **External Hackathons** — Fully managed, brand-building hackathons for talent sourcing
9. **Internal Hackathons** — Employee innovation challenges

### UPSKILL
10. **Learning & Development** — Role-based skill assessments, tailored learning paths, AI-driven
11. **AI Practice Agent** — Personal training companion for developers (mock interviews, coding practice)

| Dimension | HackerEarth | HackerRank |
|---|---|---|
| Developer community | 10M+ (sourcing advantage) | Large but more fragmented |
| Hackathons / sourcing | Full managed service | Limited |
| AI Interviewer | Yes (OnScreen, 24/7) | Limited |
| Customization | High — real-world simulations | Moderate |
| Analytics | Detailed session replay, question analytics | Robust dashboards |
| **Win angle** | "HackerEarth does sourcing + screening + interviewing in one platform — HackerRank is just assessment" | |

| Dimension | HackerEarth | TestGorilla |

| Live interviews | FaceCode + AI Interviewer | No live interview tool |
| **Win angle** | "TestGorilla is built for general hiring — HackerEarth is purpose-built for engineering teams who need real technical signal" | |

| Dimension | HackerEarth | CodeSignal |
